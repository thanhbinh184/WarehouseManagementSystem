Xây dựng hệ thống quản lý kho đặc thù cho điện thoại/laptop dựa trên mã IMEI.
Phát triển RESTful API bằng FastAPI, tích hợp trí tuệ nhân tạo (Gemini AI) để truy vấn dữ liệu và dự báo nhập hàng.
Thiết kế giao diện Responsive bằng ReactJS và Tailwind CSS, hỗ trợ quét mã vạch trực tiếp trên trình duyệt.
